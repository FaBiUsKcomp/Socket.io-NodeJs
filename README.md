# Integração NodeJs + Socket.io ![alt text](https://miro.medium.com/max/2792/1*xx7Ae6QSUV47j8DmrLCPgg.png)

## Descrição:

#### Nesse repositório consta uma pequena aplição de Chat (Simples) construída com NodeJs integrado com Socket.io
A aplição possui as seguintes funcionalidades:

- Notificação de Usuário Online.
- Chat de Mensagens Simples.
- Formatação de mensagens.

#### Ferramentas usadas:
#### Frontend:
- Html, Css e Js vanilla.
#### Backend:
- NodeJs e Socket.io
